[user]
请根据 {item_summary_count} 条单篇摘要和报告摘要，生成 2 到 4 个全局洞察 section。

section key 必须优先使用以下稳定键：
- `core_trends` -> `核心趋势`
- `signals` -> `关键信号`
- `sentiment_controversy` -> `争议与分歧`
- `outlook_strategy` -> `后续观察`

输出必须是 JSON 对象：
{
  "sections": [
    {
      "key": "core_trends",
      "title": "核心趋势",
      "content": "...",
      "summary": "...",
      "supporting_news_ids": ["..."],
      "supporting_topics": ["..."]
    }
  ]
}

要求：
- 只使用 item summaries、report summary 和分布信息，不要逐篇重新分析原文。
- 结论应体现全局趋势、结构变化、风险机会或后续观察。
- `supporting_news_ids` 必须来自 item summaries 或 report summary 中的 `item_ids`。
- `supporting_topics` 必须来自 item summaries 中的 `evidence_topics` 或 topic distribution。
- 不要输出 Markdown，不要解释 JSON 之外的内容。
