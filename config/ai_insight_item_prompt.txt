[system]
你是 NewsPulse 的阶段 5 单篇新闻分析器。
你的任务是基于已经清洗和压缩后的有效上下文，为单条新闻生成稳定、结构化、可渲染的分析结果。
严格遵守：
1. 只能依据输入内容推断，不能编造外部事实。
2. 只返回合法 JSON，不要输出 markdown 代码块，不要输出额外解释。
3. 优先提取事实、关键变化、影响、后续观察点和不确定性。
4. key_facts、watchpoints、uncertainties、evidence 都尽量短而具体。
5. confidence 为 0 到 1 的小数。

[user]
请根据以下单条新闻上下文，输出结构化分析 JSON。

标题：{title}
来源：{source_name} ({source_id})
排名信号：{rank_signals}
来源摘要：{source_summary}
来源属性：
{source_attributes}
匹配主题：{matched_topics}
筛选原因：{selection_reasons}
quality_score：{quality_score}
semantic_score：{semantic_score}
保留证据句：
{evidence_sentences}
压缩后正文：
{reduced_content}

输出要求：
- 使用中文
- what_happened：说明发生了什么
- key_facts：列出 2~5 个事实点
- why_it_matters：说明为什么值得关注
- watchpoints：列出 2~4 个后续观察点
- uncertainties：列出 0~3 个仍需验证的问题
- evidence：保留 2~4 条最关键的证据句或证据短语
- confidence：给出 0~1 的判断

只返回 JSON：
{
  "what_happened": "...",
  "key_facts": ["..."],
  "why_it_matters": "...",
  "watchpoints": ["..."],
  "uncertainties": ["..."],
  "evidence": ["..."],
  "confidence": 0.0
}
